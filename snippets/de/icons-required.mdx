<ResponseField name="icon" type="string" required>
  Das anzuzeigende Icon.
  
  Optionen:
  - Name des [Font Awesome Icon](https://fontawesome.com/icons)
  - Name des [Lucide Icon](https://lucide.dev/icons)
  - JSX-kompatibler SVG-Code in geschweiften Klammern
  - URL zu einem extern gehosteten Icon
  - Pfad zu einer Icon-Datei in Ihrem Projekt
  
  Für benutzerdefinierte SVG-Icons:
  1. Konvertieren Sie Ihr SVG mit dem [SVGR-Konverter](https://react-svgr.com/playground/).
  1. Fügen Sie Ihren SVG-Code in das SVG-Eingabefeld ein.
  2. Kopieren Sie das vollständige `<svg>...</svg>`-Element aus dem JSX-Ausgabefeld.
  3. Umschließen Sie den JSX-kompatiblen SVG-Code mit geschweiften Klammern: `icon={<svg ...> ... </svg>}`.
  4. Passen Sie `height` und `width` bei Bedarf an.
</ResponseField>

<ResponseField name="iconType" type="string">
  Der Icon-Stil von [Font Awesome](https://fontawesome.com/icons). Wird nur mit Font Awesome-Icons verwendet. 
  
  Optionen: `regular`, `solid`, `light`, `thin`, `sharp-solid`, `duotone`, `brands`.
</ResponseField>
