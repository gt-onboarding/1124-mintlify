<div id="rate-limits">
  ## Ratenlimits
</div>

Die Assistant-API hat die folgenden Limits:

- 10.000 Nutzungen pro key und Monat
- 10.000 Anfragen pro Mintlify-Organisation und Stunde
- 10.000 Anfragen pro IP-Adresse und Tag

<div id="suggested-usage">
  ## Empfohlene Verwendung
</div>

Für beste Ergebnisse verwenden Sie den [useChat-Hook aus ai-sdk](https://ai-sdk.dev/docs/reference/ai-sdk-ui/use-chat#usechat), um Anfragen zu senden und Antworten zu verarbeiten.

Sie können `fp`, `threadId` und `filter` im `body`-Feld des an den Hook übergebenen Options-Parameters festlegen.
