<div id="adding-code-samples">
  ## Hinzufügen von Codebeispielen
</div>

Sie können Inline-Code-Snippets oder Code-Blöcke hinzufügen. Code-Blöcke unterstützen Metaoptionen für Syntaxhervorhebung, Titel, Zeilenhervorhebung, Icons und mehr.

<div id="inline-code">
  ### Inline-Code
</div>

Um ein `Wort` oder eine `Wortgruppe` als Code zu kennzeichnen, umschließe es mit Backticks (`).

```mdx
Um ein `Wort` oder einen `Ausdruck` als Code zu kennzeichnen, schließen Sie es in Backticks (`) ein.
```

<div id="code-blocks">
  ### Code-Blocks
</div>

Verwenden Sie [fenced Code-Blocks](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks), indem Sie Code in drei Backticks einschließen. Code-Blocks sind kopierbar, und wenn der Assistent aktiviert ist, können Nutzer die KI bitten, den Code zu erklären.

Geben Sie die Programmiersprache für die Syntaxhervorhebung an und aktivieren Sie Meta-Optionen. Fügen Sie nach der Sprache beliebige Meta-Optionen wie einen Titel oder ein Icon hinzu.

```java HelloWorld.java lines icon="java"
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hallo, Welt!");
    }
}
```

````mdx
```java HelloWorld.java lines icon="java"
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hallo, Welt!");
    }
}
```
````

<div id="code-block-options">
  ## Code-Block-Optionen
</div>

Sie können Meta-Optionen zu Ihren Code-Blocks hinzufügen, um deren Darstellung anzupassen.

<Note>
  Bevor Sie weitere Meta-Optionen hinzufügen, müssen Sie für einen Code-Block eine Programmiersprache angeben.
</Note>

<div id="option-syntax">
  ### Optionsyntax
</div>

* **String- und Boolean-Optionen**: Mit `""`, `''` oder ohne Anführungszeichen einschließen.
* **Expressions-Optionen**: Mit `{}`, `""` oder `''` einschließen.

<div id="syntax-highlighting">
  ### Syntaxhervorhebung
</div>

Aktivieren Sie die Syntaxhervorhebung, indem Sie die Programmiersprache nach den öffnenden Backticks eines Code-Blocks angeben.

Wir verwenden [Shiki](https://shiki.style/) für die Syntaxhervorhebung und unterstützen alle verfügbaren Sprachen. Die vollständige Liste der [Sprachen](https://shiki.style/languages) finden Sie in der Shiki-Dokumentation.

Passen Sie Code-Block-Themes global über `styling.codeblocks` in Ihrer `docs.json` an. Legen Sie einfache Themes wie `system` oder `dark` fest oder konfigurieren Sie benutzerdefinierte [Shiki-Themes](https://shiki.style/themes) für light- und dark-Modus. Siehe [Settings](/de/settings#styling) für ausführliche Konfigurationsoptionen.

```java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hallo, Welt!");
    }
}
```

````mdx
```java

    public static void main(String[] args) {

    }
````

<div id="expandable">
  ### Ausklappbar
</div>

Ermöglichen Sie Benutzer:innen, lange Code-Blocks mit `expandable` ein- und auszuklappen.

```python Expandable Example expandable
from datetime import datetime, timedelta
from typing import Dict, List, Optional
from dataclasses import dataclass

@dataclass
class Book:
    title: str
    author: str
    isbn: str
    checked_out: bool = False
    due_date: Optional[datetime] = None

class Library:
    def __init__(self):
        self.books: Dict[str, Book] = {}
        self.checkouts: Dict[str, List[str]] = {}  # Nutzer -> Liste von ISBNs

    def add_book(self, book: Book) -> None:
        if book.isbn in self.books:
            raise ValueError(f"Buch mit ISBN {book.isbn} ist bereits vorhanden")
        self.books[book.isbn] = book

    def checkout_book(self, isbn: str, patron: str, days: int = 14) -> None:
        if patron not in self.checkouts:
            self.checkouts[patron] = []

        book = self.books.get(isbn)
        if not book:
            raise ValueError("Buch nicht gefunden")

        if book.checked_out:
            raise ValueError("Buch ist bereits ausgeliehen")

        if len(self.checkouts[patron]) >= 3:
            raise ValueError("Nutzer hat das Ausleih-Limit erreicht")

        book.checked_out = True
        book.due_date = datetime.now() + timedelta(days=days)
        self.checkouts[patron].append(isbn)

    def return_book(self, isbn: str) -> float:
        book = self.books.get(isbn)
        if not book or not book.checked_out:
            raise ValueError("Buch nicht gefunden oder nicht ausgeliehen")

        late_fee = 0.0
        if datetime.now() > book.due_date:
            days_late = (datetime.now() - book.due_date).days
            late_fee = days_late * 0.50

        book.checked_out = False
        book.due_date = None

        # Remove from patron's checkouts
        # Aus den Ausleihen des Nutzers entfernen
            if isbn in books:
                books.remove(isbn)
                break

        return late_fee

    def search(self, query: str) -> List[Book]:
        query = query.lower()
        return [
            book for book in self.books.values()
            if query in book.title.lower() or query in book.author.lower()
        ]

def main():
    library = Library()

    # Einige Bücher hinzufügen
    books = [
        Book("The Hobbit", "J.R.R. Tolkien", "978-0-261-10295-4"),
        Book("1984", "George Orwell", "978-0-452-28423-4"),
    ]

    for book in books:
        library.add_book(book)

    # Beispiel für Ausleihe und Rückgabe
    library.checkout_book("978-0-261-10295-4", "patron123")
    late_fee = library.return_book("978-0-261-10295-4")
    print(f"Verspätungsgebühr: ${late_fee:.2f}")

if __name__ == "__main__":
    main()
```

<div id="diff">
  ### Diff
</div>

Zeigen Sie einen visuellen Diff von hinzugefügten oder entfernten Zeilen in Ihren Code-Blöcken. Hinzugefügte Zeilen werden grün, entfernte Zeilen rot hervorgehoben.

Um Diffs zu erstellen, fügen Sie diese speziellen Kommentare am Zeilenende in Ihrem Code-Block hinzu:

* `// [!code ++]`: Markiert eine Zeile als hinzugefügt (grün hervorgehoben).
* `// [!code --]`: Markiert eine Zeile als entfernt (rot hervorgehoben).

Für mehrere aufeinanderfolgende Zeilen geben Sie die Anzahl der Zeilen nach einem Doppelpunkt an:

* `// [!code ++:3]`: Markiert die aktuelle Zeile plus die nächsten zwei Zeilen als hinzugefügt.
* `// [!code --:5]`: Markiert die aktuelle Zeile plus die nächsten vier Zeilen als entfernt.

Die Kommentarsyntax muss zu Ihrer Programmiersprache passen (zum Beispiel `//` für JavaScript oder `#` für Python).

```js Diff Example icon="code" lines
const greeting = "Hallo, Welt!"; // [!code ++]
function sayHello() {
  console.log("Hallo, Welt!"); // [!code --]
  console.log(greeting); // [!code ++]
}
sayHello();
```
