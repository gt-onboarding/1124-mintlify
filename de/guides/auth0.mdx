<Note>
  **Sicherheitshinweis**: Diese Anleitung unterstützt Sie bei der Integration von Auth0 mit Mintlify. Bitte konsultieren Sie Ihr Sicherheitsteam, bevor Sie eine Authentifizierungslösung implementieren. Mintlify übernimmt keine Verantwortung für Sicherheitsprobleme, die aus Ihrer spezifischen Implementierung entstehen können.
</Note>

<div id="overview">
  ## Überblick
</div>

Diese Anleitung führt Sie durch die Einrichtung von Auth0 als Authentifizierungsanbieter für Ihre Mintlify-Dokumentation. Am Ende können sich Ihre Nutzer mit ihren Auth0-Zugangsdaten bei Ihrer Dokumentation anmelden.

<Steps>
  <Step title="Create a Regular Web Application in Auth0">
    Melden Sie sich bei Ihrem Auth0-Dashboard an und navigieren Sie zu **Applications** > **Applications**. Klicken Sie auf **Create Application**, geben Sie Ihrer Anwendung einen Namen (zum Beispiel „Mintlify“) und wählen Sie **Regular Web Applications** als Anwendungstyp. Klicken Sie anschließend auf **Create**.
    
    <Frame>![Erstellen einer neuen Anwendung in Auth0](/images/guides/auth0/auth0-create-application.png)</Frame>
  </Step>
  <Step title="Get client information">
    Nachdem Sie Ihre Anwendung erstellt haben, gelangen Sie zur Einstellungsseite der Anwendung. Dort finden Sie die benötigten Zugangsdaten für die OAuth-Integration:
    
    <Frame>![Auth0-Client-Einstellungsseite](/images/guides/auth0/auth0clientsettings.png)</Frame>
    
    Notieren Sie sich die folgenden Informationen:
    - **Domain**: Ihre Auth0-Tenant-Domain (zum Beispiel `your-tenant.auth0.com`)
    - **Client ID**: Der öffentliche Bezeichner Ihrer Anwendung
    - **Client Secret**: Der geheime key Ihrer Anwendung (sicher aufbewahren)
    
    Diese Werte benötigen Sie zur Konfiguration von Mintlify im nächsten Schritt.
  </Step>
  <Step title="Setup Mintlify client">
    Navigieren Sie zu Ihrem Mintlify-Dashboard und gehen Sie zu **Settings** > **Authentication**. Wählen Sie **OAuth** als Authentifizierungsmethode. Daraufhin erscheint das OAuth-Konfigurationsformular:
    
    <Frame>![Mintlify OAuth-Clienteinstellungen](/images/guides/auth0/mintlifyoauthclientsettings.png)</Frame>
    
    Füllen Sie das Formular mit folgenden Werten aus:
    
    - **Authorization URL**: `https://YOUR_AUTH0_DOMAIN/authorize` (ersetzen Sie `YOUR_AUTH0_DOMAIN` durch Ihre Auth0-Domain aus Schritt 2)
    - **Client ID**: Geben Sie die Client ID Ihrer Auth0-Anwendung ein
    - **Client Secret**: Geben Sie das Client Secret Ihrer Auth0-Anwendung ein
    - **Scopes**: Leer lassen, sofern Sie keine benutzerdefinierten Scopes in Auth0 eingerichtet haben
    - **Token URL**: `https://YOUR_AUTH0_DOMAIN/oauth/token` (ersetzen Sie `YOUR_AUTH0_DOMAIN` durch Ihre Auth0-Domain)
    
    Nachdem Sie diese Details ausgefüllt haben, klicken Sie auf **Save changes**, um Ihre OAuth-Konfiguration zu speichern.
  </Step>
  <Step title="Configure Callback URL">
    Mintlify generiert eine eindeutige Redirect-URL, die Auth0 für einen korrekt funktionierenden OAuth-Flow kennen muss.
    
    Kopieren Sie die Redirect-URL aus den Authentication-Einstellungen Ihres Mintlify-Dashboards:
    <Frame>![Mintlify-Redirect-URL](/images/guides/auth0/mintlifyredirecturl.png)</Frame>
    
    Kehren Sie zur Einstellungsseite Ihrer Auth0-Anwendung zurück, scrollen Sie zum Abschnitt **Application URIs** und fügen Sie die Redirect-URL in das Feld **Allowed Callback URLs** ein:
    <Frame>![Auth0-Redirect-URL-Konfiguration](/images/guides/auth0/auth0redirecturl.png)</Frame>
    
    Klicken Sie unten auf der Auth0-Seite auf **Save Changes**, um diese Konfiguration zu übernehmen.
  </Step>
</Steps>
