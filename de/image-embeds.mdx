Fügen Sie Ihrer Dokumentation Bilder hinzu, betten Sie Videos ein und integrieren Sie interaktive Inhalte mit iframes.

Alle statischen Assets in Ihrem Docs-Repository werden automatisch unter dem entsprechenden Pfad auf Ihrer domain ausgeliefert. Beispiel: Wenn Sie `/images/my-logo.png` in Ihrem Repo haben, ist das Bild unter `https://docs.yoursite.com/images/my-logo.png` verfügbar.

<div id="images">
  ## Bilder
</div>

Fügen Sie Ihrer Dokumentation Bilder hinzu, um visuellen Kontext, Beispiele oder dekorative Elemente bereitzustellen.

<div id="basic-image-syntax">
  ### Grundlegende Bildsyntax
</div>

Verwenden Sie die [Markdown-Syntax](https://www.markdownguide.org/basic-syntax/#images), um Ihrer Dokumentation Bilder hinzuzufügen:

```mdx
![Alt-Text zur Bildbeschreibung](/path/to/image.png)
```

<Tip>
  Fügen Sie immer einen aussagekräftigen Alt-Text hinzu, um die Barrierefreiheit und die SEO zu verbessern. Der Alt-Text sollte klar beschreiben, was auf dem Bild zu sehen ist.
</Tip>

Bilddateien dürfen maximal 20 MB groß sein. Größere Dateien sollten Sie auf einem CDN-Dienst wie [Amazon S3](https://aws.amazon.com/s3) oder [Cloudinary](https://cloudinary.com) hosten.

<div id="html-image-embeds">
  ### HTML-Bildeinbettungen
</div>

Für mehr Kontrolle über die Bilddarstellung verwenden Sie HTML-`<img>`-Tags:

```html
<img 
  src="/images/dashboard.png" 
  alt="Haupt-Dashboard-Benutzeroberfläche"
  height="300"
  className="rounded-lg"
/>
```

<div id="disable-zoom-functionality">
  #### Zoomfunktion deaktivieren
</div>

Um das standardmäßige Zoomen per Klick für Bilder zu deaktivieren, fügen Sie die Eigenschaft `noZoom` hinzu:

```html highlight="4"
<img 
  src="/images/screenshot.png" 
  alt="Beschreibender Alt-Text"
  noZoom 
  height="200"
/>
```

<div id="link-images">
  #### Bilder verlinken
</div>

Um ein Bild klickbar zu machen, umgeben Sie es mit einem Anchor-Tag und fügen Sie die Eigenschaft `noZoom` hinzu:

```html
<a href="https://mintlify.com" target="_blank">
  <img 
    src="/images/logo.png" 
    alt="Mintlify-Logo"
    noZoom 
    height="100"
  />
</a>
```

<Note>
  Bilder innerhalb von Anchor-Tags zeigen automatisch einen Zeiger-Cursor an, der ihre Klickbarkeit signalisiert.
</Note>

<div id="light-and-dark-mode-images">
  #### Bilder für light- und Dark-Mode
</div>

Um unterschiedliche Bilder für light- und Dark-Themes anzuzeigen, verwenden Sie Tailwind-CSS-Klassen:

```html
<!-- Light mode image -->
<img 
  className="block dark:hidden" 
  src="/images/light-mode.png" 
  alt="Light-Modus-Benutzeroberfläche"
/>

<!-- Dark mode image -->
<img 
  className="hidden dark:block" 
  src="/images/dark-mode.png" 
  alt="Dark-Modus-Benutzeroberfläche"
/>
```

<div id="videos">
  ## Videos
</div>

Mintlify unterstützt [HTML-Tags in Markdown](https://www.markdownguide.org/basic-syntax/#html) und gibt Ihnen die Flexibilität, vielfältige Inhalte zu erstellen.

<Tip>
  Fügen Sie in Videoelementen stets einen Fallback-Text ein, für Browser, die keine Videowiedergabe unterstützen.
</Tip>

<div id="youtube-embeds">
  ### YouTube-Einbettungen
</div>

Bette YouTube-Videos mithilfe von iframe-Elementen ein:

```html
<iframe
  className="w-full aspect-video rounded-xl"
  src="https://www.youtube.com/embed/4KzFe50RQkQ"
  title="YouTube-Videoplayer"
  frameBorder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowFullScreen
></iframe>
```

<div id="self-hosted-videos">
  ### Selbstgehostete Videos
</div>

Verwenden Sie das HTML-Element `<video>` für selbst gehostete Videoinhalte:

```html
<video
  controls
  className="w-full aspect-video rounded-xl"
  src="link-zu-ihrem-video.com"
></video>
```

<div id="autoplay-videos">
  ### Videos automatisch abspielen
</div>

Um ein Video automatisch abzuspielen, verwenden Sie:

```html
<video
  autoPlay
  muted
  loop
  playsInline
  className="w-full aspect-video rounded-xl"
  src="/videos/demo.mp4"
></video>
```

<Note>
  Bei der Verwendung der JSX-Syntax sollten mehrteilige Attribute in camelCase geschrieben werden: `autoPlay`, `playsInline`, `allowFullScreen`.
</Note>

<div id="iframes">
  ## iframes
</div>

Binden Sie externe Inhalte mithilfe von iframe-Elementen ein:

```html
<iframe 
  src="https://example.com/embed" 
  title="Eingebetteter Inhalt"
  className="w-full h-96 rounded-xl"
></iframe>
```
