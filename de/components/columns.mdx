Die `Columns`-Komponente ermöglicht es, mehrere `Card`-Komponenten zu gruppieren. Sie wird meist verwendet, um Karten in einem Raster anzuordnen, indem die Anzahl der Spalten festgelegt wird.

<Columns cols={2}>
  <Card title="Erste Schritte" icon="rocket">
    Richten Sie Ihr Projekt mit unserer Schnellstartanleitung ein.
  </Card>

  <Card title="API-Referenz" icon="code">
    Erkunden Sie Endpoints, Parameter und Beispiele für Ihre API.
  </Card>
</Columns>

```mdx Columns example
<Columns cols={2}>
  <Card title="Erste Schritte">
    Richten Sie Ihr Projekt mit unserem Schnellstart-Leitfaden ein.
  </Card>
  <Card title="API-Referenz">
    Erkunden Sie Endpoints, Parameter und Beispiele für Ihre API.
  </Card>
</Columns>
```


<div id="properties">
  ## Eigenschaften
</div>

<ResponseField name="cols" default={2}>
  Die Anzahl der Spalten pro Zeile.
</ResponseField>
