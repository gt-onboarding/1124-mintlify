Verwenden Sie Tabs, um Inhalte in mehrere Bereiche zu gliedern, zwischen denen Nutzer wechseln können. Sie können beliebig viele Tabs hinzufügen und innerhalb jedes Tabs weitere Komponenten einfügen.

<Tabs>
  <Tab title="Erster Tab">
    ☝️ Willkommen bei den Inhalten, die nur im ersten Tab sichtbar sind.

    Sie können beliebig viele Komponenten innerhalb von Tabs hinzufügen. Zum Beispiel ein Codeblock:

    ```java HelloWorld.java
      class HelloWorld {
          public static void main(String[] args) {
              System.out.println("Hello, World!");
          }
      }
    ```
  </Tab>

  <Tab title="Zweiter Tab" icon="leaf">
    ✌️ Hier sind Inhalte, die nur im zweiten Tab enthalten sind.

    Dieser hat ein <Icon icon="leaf" /> Icon!
  </Tab>

  <Tab title="Dritter Tab">
    💪 Hier sind Inhalte, die nur im dritten Tab enthalten sind.
  </Tab>
</Tabs>

````mdx Tabs example
<Tabs>
  <Tab title="Erster Tab">
    ☝️ Willkommen zu dem Inhalt, den Sie nur innerhalb des ersten Tabs sehen können.

    Sie können beliebig viele Komponenten innerhalb von Tabs hinzufügen. Zum Beispiel einen Codeblock:
    ```java HelloWorld.java
      class HelloWorld {
          public static void main(String[] args) {
              System.out.println("Hallo, Welt!");
          }
      }
    ```
  </Tab>
  <Tab title="Zweiter Tab" icon="leaf">
    ✌️ Hier ist Inhalt, der nur innerhalb des zweiten Tabs ist.

    Dieser hat ein <Icon icon="leaf" /> Icon!
  </Tab>
  <Tab title="Dritter Tab">
    💪 Hier ist Inhalt, der nur innerhalb des dritten Tabs ist.
  </Tab>
</Tabs>
````


<div id="properties">
  ## Eigenschaften
</div>

<ResponseField name="title" type="string" required>
  Der Titel des Tabs. Kurze Titel erleichtern die Navigation.
</ResponseField>

<ResponseField name="icon" type="string">
  Ein [Font Awesome](https://fontawesome.com/icons) Icon, [Lucide](https://lucide.dev/icons) Icon, eine URL zu einem Icon oder ein relativer Pfad zu einem Icon.
</ResponseField>

<ResponseField name="iconType" type="string">
  Nur für Font Awesome Icons: Eines von `regular`, `solid`, `light`, `thin`, `sharp-solid`, `duotone`, `brands`.
</ResponseField>
