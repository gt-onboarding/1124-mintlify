---
title: "Panel"
description: "Den Inhalt des rechten Seitenpanels festlegen"
icon: "panel-right"
---

Mit der `<Panel>`-Komponente können Sie das rechte Seitenpanel einer Seite mit beliebigen Komponenten anpassen.

Wenn eine Seite eine `<Panel>`-Komponente enthält, müssen alle [RequestExample](/de/components/examples#request-example)- und [ResponseExample](/de/components/examples#response-example)-Komponenten innerhalb von `<Panel>` platziert werden.

Die Komponenten in einem `<Panel>` ersetzen das Inhaltsverzeichnis der Seite.

```mdx
<Panel>
  <Info>Heften Sie Infos an das Seitenpanel an. Oder fügen Sie eine andere Komponente hinzu.</Info>
</Panel>
```

<Panel>
  <Info>Info an die Seitenleiste anheften. Oder eine andere Komponente hinzufügen.</Info>
</Panel>
