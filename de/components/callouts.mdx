---
title: "Callouts"
description: "Verwenden Sie Callouts, um Ihrem Inhalt auffälligen Kontext hinzuzufügen"
icon: "info"
---

Callouts können als Note, Warning, Info, Hinweis, Check oder Danger gestaltet werden, oder Sie erstellen Ihr eigenes Callout:

<Note>Dies fügt dem Inhalt eine Notiz hinzu</Note>

```mdx
<Note>Dies fügt eine Notiz in den Inhalt ein</Note>
```

<Warning>Dies löst eine Warning aus, auf die man achten sollte.</Warning>

```mdx
<Warning>Dies löst eine Warnung aus, die beachtet werden sollte</Warning>
```

<Info>Dies weist auf wichtige Infos hin</Info>

```mdx
<Info>Dies macht auf wichtige Informationen aufmerksam</Info>
```

<Tip>Dies ist ein hilfreicher Hinweis</Tip>

```mdx
<Tip>Dies zeigt einen hilfreichen Hinweis</Tip>
```

<Check>Dadurch erhalten wir den Status „Checked“</Check>

```mdx
<Check>Dies zeigt uns einen geprüften Status</Check>
```

<Danger>Dies ist ein Gefahrenhinweis</Danger>

```mdx
<Danger>Dies ist ein Warnhinweis</Danger>
```

<Callout icon="key" color="#FFC107" iconType="regular"> Dies ist ein benutzerdefiniertes Hinweisfeld</Callout>

```mdx wrap
<Callout icon="key" color="#FFC107" iconType="regular">Dies ist ein benutzerdefiniertes Callout</Callout>
```
