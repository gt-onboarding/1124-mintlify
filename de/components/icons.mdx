Verwenden Sie Icons aus Font Awesome, Lucide, SVGs, externen URLs oder Dateien in Ihrem Projekt, um Ihre Dokumentation zu verbessern.

<Icon icon="flag" size={32} />

```mdx Icon example
<Icon icon="flag" size={32} />
```


<div id="inline-icons">
  ## Inline-Icons
</div>

Icons werden inline platziert, wenn sie in einem Absatz verwendet werden.

<Icon icon="flag" iconType="solid" /> Die gewünschte Dokumentation – mühelos.

```markdown Inline icon example
<Icon icon="flag" iconType="solid" /> Die Dokumentation, die Sie brauchen, mühelos.
```

  Die Größe des Icons in Pixel.
