Automatisch generierte Code-Snippets aus Speakeasy-SDKs lassen sich direkt in die Mintlify-API-Referenzdokumentation integrieren. SDK-Beispielsnippets werden im [interaktiven Playground](/de/api-playground/overview) auf Mintlify-basierten Dokumentationsseiten angezeigt.

<Frame>
  ![Ein Mintlify API Playground mit Speakeasy-Code-Snippets.](/images/speakeasy/mintlify-with-speakeasy-openapi.png)
</Frame>

<div id="prerequisites">
  ## Voraussetzungen
</div>

Um Mintlify in Speakeasy zu integrieren, benötigen Sie Folgendes:

* Ein [Repository für Mintlify-Dokumentation](/de/quickstart#creating-the-repository).
  * Ein von Speakeasy generiertes SDK mit einer konfigurierten [automatisierten Codebeispiel-URL](https://www.speakeasy.com/docs/code-samples/automated-code-sample-urls).

<div id="setting-up-the-integration">
  ## Einrichten der Integration
</div>

Um Speakeasy mit Mintlify zu integrieren, müssen Sie die öffentliche URL der kombinierten API-Spezifikation aus dem Registry abrufen und Ihre Konfigurationsdatei `docs.json` aktualisieren.

<div id="get-the-apis-combined-spec-public-url-from-the-registry">
  ### Rufen Sie die öffentliche URL der kombinierten API-Spezifikation aus dem Registry ab
</div>

Navigieren Sie zu Ihrem [Speakeasy Dashboard](https://app.speakeasy.com) und öffnen Sie den Tab **API Registry**. Öffnen Sie den Eintrag `*-with-code-samples` für die API.

<Frame>
  ![Screenshot der Speakeasy-API-Registry-Seite. Der Tab API Registry ist mit einem roten Quadrat und der Nummer 1 hervorgehoben, und der Eintrag für die API ist mit einem roten Quadrat und der Nummer 2 hervorgehoben.](/images/speakeasy/openapi-registry-and-combined-spec.png)
</Frame>

<Note>
  Wenn der Eintrag nicht mit **Combined Spec** gekennzeichnet ist, stellen Sie sicher, dass für die API eine [automatische Codebeispiel-URL](https://www.speakeasy.com/docs/code-samples/automated-code-sample-urls) konfiguriert ist.
</Note>

Kopieren Sie auf der Seite des Registry-Eintrags die bereitgestellte öffentliche URL.

<Frame>
  ![Screenshot, der den Registry-Eintrag der kombinierten Spezifikation zeigt, wobei die Funktion zum Kopieren der URL mit einem roten Quadrat hervorgehoben ist.](/images/speakeasy/copy-combined-spec-url.png)
</Frame>

<div id="update-your-docsjson-configuration-file">
  ### Aktualisieren Sie Ihre `docs.json`-Konfigurationsdatei
</div>

Fügen Sie die kombinierte Spec-URL in Ihrer `docs.json`-Datei einem Abschnitt **Anchors** oder **Tabs** hinzu.

Fügen Sie die kombinierte Spec-URL einem Anchor hinzu, indem Sie das Feld `anchor` in Ihrer `docs.json`-Datei wie folgt aktualisieren:

```json docs.json
{
  "anchors": [
    {
      "name": "API-Referenz",
      // !mark
      "openapi": "SPEAKEASY_COMBINED_SPEC_URL",
      "url": "api-reference",
      "icon": "square-terminal"
    }
  ]
}
```

Fügen Sie die kombinierte Spec-URL zu einem Tab hinzu, indem Sie das Feld `tab` in der Datei `docs.json` wie folgt aktualisieren:

```json docs.json
{
  "tabs": [
    {
      "name": "API-Referenz",
      "url": "api-reference",
      // !mark
      "openapi": "SPEAKEASY_COMBINED_SPEC_URL"
    }
  ]
}
```

Von Speakeasy generierte Codebeispiele können jetzt in Ihren API-Dokumentationen angezeigt und im Playground ausprobiert werden.
