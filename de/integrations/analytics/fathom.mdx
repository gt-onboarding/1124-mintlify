---
title: "Fathom"
---

Fügen Sie Folgendes zu Ihrer Datei `docs.json` hinzu, um Analytics an Fathom zu senden.

Sie können die `siteId` in Ihren Skripteinstellungen finden.

<CodeGroup>

```json Analytics-Optionen in docs.json
"integrations": {
    "fathom": {
        "siteId": "required"
    }
}
```

```json Beispiel
"integrations": {
    "fathom": {
        "siteId": "YSVMSDAY"
    }
}
```

</CodeGroup>
