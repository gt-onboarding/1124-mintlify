---
title: "Google Tag Manager"
---

Fügen Sie Ihre Tag-ID in die Datei `docs.json` ein, dann betten wir das Google-Tag-Manager-Skript auf all Ihren Seiten ein.

Sie sind dafür verantwortlich, bei Bedarf Cookie-Einwilligungsbanner mit dem Google Tag Manager einzurichten.

<CodeGroup>

```json Analytics-Optionen in docs.json
"integrations": {
    "gtm": {
        "tagId": "required"
    }
}
```

```json Beispiel
"integrations": {
    "gtm": {
        "tagId": "GTM-MGBL4PW"
    }
}
```

</CodeGroup>
