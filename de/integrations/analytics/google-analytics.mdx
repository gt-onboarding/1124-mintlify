title: "Google Analytics 4"
---

Sie müssen eine neue <Tooltip tip="Google Analytics 4">GA4</Tooltip>-Property erstellen, um sie mit Mintlify zu verwenden. Die erfassten Daten fließen in dasselbe Projekt wie Ihre übrigen Google-Analytics-Daten.

Wenn Sie die frühere Version von Google Analytics (Universal Analytics) verwenden, können Sie dennoch eine <Tooltip tip="Google Analytics 4">GA4</Tooltip>-Property erstellen. <Tooltip tip="Google Analytics 4">GA4</Tooltip>-Daten unterscheiden sich leicht von UA-Daten, werden jedoch im selben Projekt erfasst.

<div id="how-to-connect-ga4-to-mintlify">
  ## GA4 mit Mintlify verbinden
</div>

<div id="create-a-web-stream">
  ### Erstellen Sie einen Webstream
</div>

Sie müssen einen Webstream erstellen, um die Measurement ID zu erhalten, die Sie in Mintlify einfügen.

Klicken Sie auf das Zahnrad unten links im Google-Analytics-Bildschirm. Klicken Sie dann auf Data Streams.

<Frame>![](/images/ga4-web-streams.png)</Frame>

Erstellen Sie einen Webstream und geben Sie die URL Ihrer Mintlify-Dokumentationsseite als Stream-URL an.

Ihre Measurement ID hat das Format `G-XXXXXXX` und wird unmittelbar nach dem Erstellen des Webstreams unter Stream Details angezeigt.

<div id="put-measurement-id-in-docsjson">
  ### Measurement-ID in docs.json eintragen
</div>

Tragen Sie Ihre Measurement-ID wie folgt in die Datei `docs.json` ein:

```json docs.json
"integrations": {
    "ga4": {
        "measurementId": "G-XXXXXXX"
    }
}
```


<div id="wait">
  ### Warten
</div>

Google Analytics benötigt zwei bis drei Tage, um Ihre Daten anzuzeigen.

Sie können den [Google Analytics Debugger](https://chrome.google.com/webstore/detail/google-analytics-debugger/jnkmfdileelhofjcijamephohjechhna?hl=en) verwenden, um zu prüfen, ob Analytics korrekt aktiviert ist. Die Erweiterung protokolliert in der Browserkonsole jedes Mal, wenn GA4 eine Anfrage sendet.

<Note>

Vorschaulinks haben Analytics deaktiviert.

</Note>
