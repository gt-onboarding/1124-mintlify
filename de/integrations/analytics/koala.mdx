---
title: "Koala"
---

Fügen Sie die folgenden Einträge zu Ihrer `docs.json` hinzu, um Analytics an Koala zu senden.

<CodeGroup>

```json Analytics-Optionen in docs.json
"integrations": {
    "koala": {
        "publicApiKey": "required"
    }
}
```

```json Beispiel
"integrations": {
    "koala": {
        "publicApiKey": "pk_1a1882"
    }
}
```

</CodeGroup>
