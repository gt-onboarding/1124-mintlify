---
title: "Clearbit"
---

Fügen Sie Folgendes zu Ihrer Datei `docs.json` hinzu, um Analytics an Clearbit zu übermitteln.

<CodeGroup>

```json Analytics-Optionen in docs.json
"integrations": {
    "clearbit": {
        "publicApiKey": "required"
    }
}
```

```json Beispiel
"integrations": {
    "clearbit": {
        "publicApiKey": "pk_1a1882"
    }
}
```

</CodeGroup>
