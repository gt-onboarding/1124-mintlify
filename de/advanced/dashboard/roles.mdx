---
title: "Rollen"
description: "Steuern Sie den Zugriff auf Ihr Dashboard mit Rollen."
---

<Info>
  RBAC-Funktionalität ist im [Enterprise-Tarif](https://mintlify.com/pricing?ref=rbac) verfügbar.
</Info>

Mintlify bietet zwei Zugriffsebenen im Dashboard: Editor und Admin.

Im Folgenden sind Aktionen aufgeführt, die der Rolle Admin vorbehalten sind:

|                         | Editor | Admin |
| ----------------------- | :----: | :---: |
| Benutzerrollen aktualisieren       |   ❌   |  ✅   |
| Benutzer löschen            |   ❌   |  ✅   |
| Admins einladen      |   ❌   |  ✅   |
| Abrechnung verwalten und aktualisieren |   ❌   |  ✅   |
| Benutzerspezifische Domain aktualisieren    |   ❌   |  ✅   |
| Git-Quelle aktualisieren       |   ❌   |  ✅   |
| Organisation löschen              |   ❌   |  ✅   |

Weitere Aktionen im Dashboard sind für beide Rollen verfügbar.

Sie können beliebig viele Admins einladen, wir empfehlen jedoch, den Admin-Zugriff auf Nutzer zu beschränken, die ihn benötigen.
