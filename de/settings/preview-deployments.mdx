<Info>
  Preview-Bereitstellungen sind in den [Pro- und Enterprise-Plänen](https://mintlify.com/pricing?ref=preview-deployments) verfügbar.
</Info>

Preview-Bereitstellungen zeigen, wie sich Änderungen an Ihrer Dokumentation auswirken, bevor sie in die Produktion übernommen werden. Jede Vorschau erzeugt eine teilbare URL, die sich automatisch aktualisiert, sobald Sie neue Änderungen pushen.

<div id="creating-preview-deployments">
  ## Vorschau-Deployments erstellen
</div>

Vorschau-Deployments werden automatisch über Pull Requests oder manuell über das Dashboard erstellt.

<div id="automatic-previews">
  ### Automatische Vorschauen
</div>

Wenn du einen Pull Request erstellst, fügt der Mintlify‑Bot automatisch einen Link hinzu, über den du die Preview‑Deployment in deinem Pull Request ansehen kannst. Die Vorschau wird jedes Mal aktualisiert, wenn du neue Commits in die Branch pushst.

<div id="manual-previews">
  ### Manuelle Previews
</div>

Sie können für jede Branch manuell eine Preview erstellen.

1. Gehen Sie zu Ihrem [Dashboard](https://dashboard.mintlify.com/).
2. Wählen Sie **Previews**.
3. Wählen Sie **Create preview**.
4. Geben Sie den Namen Ihrer Branch ein.
5. Wählen Sie **Create deployment**.

<div id="redeploying-a-preview">
  ## Vorschau erneut bereitstellen
</div>

Stellen Sie eine Vorschau erneut bereit, um Inhalte zu aktualisieren oder einen fehlgeschlagenen Deployment-Vorgang erneut zu versuchen.

1. Wählen Sie die Vorschau in Ihrem [Dashboard](https://dashboard.mintlify.com/).
2. Wählen Sie **Redeploy**.

<div id="sharing-preview-deployments">
  ## Vorschau-Bereitstellungen freigeben
</div>

Vorschau-Bereitstellungen können öffentlich zugänglich sein oder auf authentifizierte Nutzer beschränkt werden.

**Öffentlicher Zugriff**: Vorschau-URLs sind standardmäßig öffentlich einsehbar. Geben Sie einen Vorschau-Link für alle frei, die Ihre Änderungen prüfen müssen.

**Eingeschränkter Zugriff**: Aktivieren Sie die Authentifizierung für Vorschau-Bereitstellungen, um den Zugriff auf Vorschauen auf authentifizierte Mintlify-Nutzer zu beschränken. Weitere Informationen finden Sie unter [Authentication Setup](/de/authentication-personalization/authentication-setup).
