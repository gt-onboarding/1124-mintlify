<div id="ask-our-docs">
  ## Unsere Dokumentation befragen
</div>

Drücke <kbd>Command</kbd> + <kbd>I</kbd>, um einen Chat mit unserem auf unserer Dokumentation trainierten KI-Assistenten zu starten.

<div id="watch-video-tutorials">
  ## Video-Tutorials ansehen
</div>

Besuchen Sie unseren [YouTube](https://www.youtube.com/@GetMintlify/videos)-Kanal für Tutorials und Anleitungen zur Nutzung von Mintlify.

<div id="message-support">
  ## Nachrichtensupport
</div>

Senden Sie uns eine Nachricht über Ihr [Dashboard](https://dashboard.mintlify.com/), indem Sie in der Seitenleiste **Support** auswählen.

![GIF, das zeigt, wie Sie über das Dashboard auf den Support zugreifen, indem Sie die Schaltfläche Support auswählen, anschließend eine Option wählen und eine Frage in das erscheinende Support-Modal eingeben.](https://mintlify.s3.us-west-1.amazonaws.com/mintlify/images/support-flow.gif)

<Info>
Wir bemühen uns, innerhalb von 24 Stunden auf alle Anfragen zu antworten; während Stoßzeiten kann es jedoch zu Verzögerungen kommen.
</Info>

<div id="email-support">
  ## E-Mail-Support
</div>

Wenn Sie nicht auf Ihr Dashboard zugreifen können, schreiben Sie uns bitte an <a href="mailto:support@mintlify.com">support@mintlify.com</a>.
